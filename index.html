day24 回溯算法 77.组合

发表于 2023-10-16 更新于 2023-10-18

理论基础

  • 回溯算法的本质就是穷举，穷举所有可能的结果，然后找到满足条件的结果。
  • 如果想让回溯法高校一些，可以加一些剪枝的操作，但是本质还是穷举
  • 回溯法解决的问题
    • 组合问题: N 个数里面按一定规则找到 k 个数的集合
    • 排列问题: N 个数按一定规则全排列，有几种排列方式
    • 切割问题: 一个字符串按一定规则有几种切割方式
    • 子集问题: 一个 N 个数的集合里有多少符合条件的子集
    • 棋盘问题: N 皇后，解数独等等
  • 回溯法解决的问题都可以抽象成树形结构，然后使用 DFS 遍历整棵树，每次遍历到叶子节点的时候，就得到了一个正确的结果。
  • 回溯法模板

77.组合

题目链接:77.组合

思路

每次把一个数 i 存到 cur 中，然后遍历比 i 大的数，递归。当 cur 的长度达到 k 时，存到 res 中。需要注意的就是回溯，也就是每次递归完后，需要把 cur 中的最后一个数 pop 出来，然后继续遍历。

450.删除二叉搜索树中的节点

题目链接:450.删除二叉搜索树中的节点

思路

同样的按照key值大于当前节点往右找，小于当前节点往左找，找到key值对应的节点。如果找到空节点了，说明树中没有这个要删的点，直接返回就行了。如果找到了，分类讨论:
1.当前节点是叶子节点，直接返回None
2.当前节点左子树为空，右子树不为空，那直接返回右子树就行了
3.当前节点左子树不为空，右子树为空，那直接返回左子树就行了
4.当前节点左右子树都不为空，那就让左子树变成右子树的最左边的左子树，然后返回右子树就行了

Python

时间复杂度:O(n) 空间复杂度:O(n)

236. 二叉树的最近公共祖先

题目链接:236. 二叉树的最近公共祖先

思路

这道题跟上面的两道题不一样，上面两道题都是二叉搜索树，这道题是普通的二叉树，所以不能使用中序遍历，而是使用后序遍历。后序遍历的话，先遍历左右子树，然后再遍历根节点，这样的话，如果左右子树都有p或者q，那么根节点就是最近公共祖先，如果只有左子树有p或者q，说明最近公共祖先在左子树，或者还没找到，还得再往上找，如果只有右子树有p或者q，说明最近公共祖先在右子树，或者还没找到，还得再往上找。如果左右子树都没有p或者q，说明最近公共祖先在上面，或者还没找到，还得再往上找。不管如何，返回左跟右的结果，剩下的交给上面去继续处理就行了。

Python

700.二叉搜索树中的搜索

题目链接:700.二叉搜索树中的搜索

递归思路

二叉搜索树中，左孩子一定小于父节点小于右孩子，所以碰到当前节点值比要求大，就往左子树找，碰到当前节点值比要求小，就往右子树找，碰到当前节点值等于要求，就返回当前节点。碰到空节点，就返回 None

迭代思路

一样的，使用 BFS，因为二叉搜索树的性质，不需要使用队列，直接选择往左还是往右就行了。碰到空节点，就返回 None

98.验证二叉搜索树

题目链接:98.验证二叉搜索树

递归思路

使用 DFS，因为二叉搜索树的性质，所以只需要判断当前节点是否大于左子树的最大值，小于右子树的最小值就行了。所以需要记录左子树的最大值和右子树的最小值。然后分别对左右子树进行递归。如果当前节点不满足条件，就返回 False，否则返回 True。如果当前节点是空节点，就返回 True。因为空节点肯定是二叉搜索树的一部分。

迭代思路

使用 BFS，跟递归一样的做法

中序遍历

二叉搜索树的中序遍历是递增的，所以可以使用中序遍历，然后判断是否递增就行了。这里使用了一个小技巧，就是使用一个变量记录上一个节点的值，然后跟当前节点的值进行比较。如果当前节点的值小于等于上一个节点的值，那么就不是二叉搜索树。如果是二叉搜索树，那么最后返回 True，否则返回 False。如果当前节点是空节点，那么就返回 True。因为空节点肯定是二叉搜索树的一部分。
