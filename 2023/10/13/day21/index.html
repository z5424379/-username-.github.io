235. 二叉搜索树的最近公共祖先

题目链接:235. 二叉搜索树的最近公共祖先
### 思路
同样是递归下去，从根节点往下，找到的第一个介于两个值中间的节点，就是最近公共祖先。并且可以根据二叉搜索树的性质，如果当前节点的值比两个目标都大，就往左，都小就往右。
### Python

时间复杂度:O(n) 空间复杂度:O(n)

701.二叉搜索树中的插入操作

题目链接:701.二叉搜索树中的插入操作
### 思路
二叉搜索树的插入操作，就是找到要插入的位置，然后插入就行了。如果当前节点为空，说明要插入的位置就是这里，如果当前节点值比要插入的值大，就往左子树找，如果当前节点值比要插入的值小，就往右子树找。找到空节点的时候，就插入。
### Python

时间复杂度:O(n) 空间复杂度:O(n)

450.删除二叉搜索树中的节点

题目链接:450.删除二叉搜索树中的节点

思路

同样的按照key值大于当前节点往右找，小于当前节点往左找，找到key值对应的节点。如果找到空节点了，说明树中没有这个要删的点，直接返回就行了。如果找到了，分类讨论:
1.当前节点是叶子节点，直接返回None
2.当前节点左子树为空，右子树不为空，那直接返回右子树就行了
3.当前节点左子树不为空，右子树为空，那直接返回左子树就行了
4.当前节点左右子树都不为空，那就让左子树变成右子树的最左边的左子树，然后返回右子树就行了

Python

时间复杂度:O(n) 空间复杂度:O(n)
