617.合并二叉树

题目链接:617.合并二叉树

思路

对两棵树一起进行递归，每个位置判断是否有对应的节点，如果两棵树当前位置都没有，那么就是空结点，如果有一个有，那么就是有的那个，如果两个都有，那么就是两个结点的值相加。然后分别对左右子树进行递归。

Python

时间复杂度: O(n)
空间复杂度: O(n)

简短写法

如果一棵树当前位置没有节点的话，那往下肯定也没有，所以直接返回另一棵树就行了。前面想多了。

700.二叉搜索树中的搜索

题目链接:700.二叉搜索树中的搜索

递归思路

二叉搜索树中，左孩子一定小于父节点小于右孩子，所以碰到当前节点值比要求大，就往左子树找，碰到当前节点值比要求小，就往右子树找，碰到当前节点值等于要求，就返回当前节点。碰到空节点，就返回 None

迭代思路

一样的，使用 BFS，因为二叉搜索树的性质，不需要使用队列，直接选择往左还是往右就行了。碰到空节点，就返回 None

98.验证二叉搜索树

题目链接:98.验证二叉搜索树

递归思路

使用 DFS，因为二叉搜索树的性质，所以只需要判断当前节点是否大于左子树的最大值，小于右子树的最小值就行了。所以需要记录左子树的最大值和右子树的最小值。然后分别对左右子树进行递归。如果当前节点不满足条件，就返回 False，否则返回 True。如果当前节点是空节点，就返回 True。因为空节点肯定是二叉搜索树的一部分。

迭代思路

使用 BFS，跟递归一样的做法

中序遍历

二叉搜索树的中序遍历是递增的，所以可以使用中序遍历，然后判断是否递增就行了。这里使用了一个小技巧，就是使用一个变量记录上一个节点的值，然后跟当前节点的值进行比较。如果当前节点的值小于等于上一个节点的值，那么就不是二叉搜索树。如果是二叉搜索树，那么最后返回 True，否则返回 False。如果当前节点是空节点，那么就返回 True。因为空节点肯定是二叉搜索树的一部分。
