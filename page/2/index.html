Python 代码:

我的大体思路是，维护两个快慢指针。如果慢指针指向的位置是要删除的元素 val 的话，那么慢指针就不动，然后用快指针指向的值覆盖慢指针位置的值。那么在最开始，如果不是 val 的话，快慢指针会指向一个位置，也就是自己替换自己。当出现第一个 val 的时候，慢指针不动，快指针继续移动，也就是不停的用后面一格的值覆盖前面一格，出现第二个 val 的时候，快慢指针相差两格，就是用后面两格的值覆盖前面的。直到快指针走到数组结束的位置停止，由于fast < len(nums)，也就是说快指针最后一次fast += 1之后会变成len(nums)，所以要加一个判断条件if fast < len(nums):防止超出范围。
